MALPY - Documentation

Description

MALPY is a python library made to help people to use both MyAnimeList API and Jikan API easily and together. Why? There are some endpoints that are available on one of them, but unavailable on the other, which forces you to use both API's if you want to access all these endpoints

If you're new with malpy, you shold start reading this: getting started

import myanimelist, asyncio

mal = myanimelist.Connect("YOUR CLIENT ID")
def main():
    search = await mal.search.anime("Death Note", limit=5)
    for anime in search:
        print(f"ID: {anime.id} - Title: {anime.title}")

asyncio.run(main())
