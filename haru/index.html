Description

This is a rework of Haru (Haru v3). A simple bot for discord, with minigames and some utilities.

  • Only works in guilds. all commands except help cannot be used in dms.
  • Modo de Desenvolvimento: funciona apenas nos servidores pré-definidos

Actual version: alpha 3.0.0

Cloning the Repository

Before copying the repository and starting the bot, a few things need to be done.

  1. Clone the repository
  2. Install the dependences using pip install -r requirements.txt
  3. Install MAL-Easy-Client by following these steps: Mal-easy-client README
  4. Create your bot in the developer portal
  5. Go to this blog and get your Client ID and Client Secret
  6. Create a .env and put the following parameters: TOKEN, CLIENT-ID and CLIENT_SECRET
  7. Start the bot with python main.py

If you want to disable Developer Mode or Debug Mode, just log into your developer server and use the following commands: h!oc_devmode off and h!oc_errorsmode off.
